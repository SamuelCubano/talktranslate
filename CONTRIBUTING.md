# Contribuyendo a TalkTranslate

Gracias por querer contribuir 🙌

## 🐛 Reportar bugs

Abre un [issue](https://github.com/SamuelCubano/talktranslate/issues) describiendo:
- Qué esperabas que pasara
- Qué pasó realmente
- Cómo reproducirlo (pasos, navegador, etc.)

## 💡 Sugerir features

Abre un issue con la etiqueta `enhancement` y describe la funcionalidad que te gustaría ver.

## 🔧 Pull Requests

1. Fork el repo
2. Crea una rama: `git checkout -b mi-feature`
3. Haz tus cambios
4. Commit: `git commit -m "Descripción clara del cambio"`
5. Push: `git push origin mi-feature`
6. Abre un Pull Request

## 📐 Estilo de código

- HTML semántico
- CSS con variables de tema (`var(--accent)`, etc.)
- JS moderno (ES6+, async/await)
- Nada de WebRTC o APIs de voz (el proyecto es solo chat)

## 🧪 Probar localmente

```bash
cd server
npm install
node server.js
# Abre http://localhost:3000
```
